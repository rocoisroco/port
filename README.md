# 🍓 SmashBerry

웹캠으로 손을 움직여 과일을 썰는 게임!  
MediaPipe 손 인식 기술을 사용한 브라우저 기반 게임입니다.

## 🎮 플레이 방법

1. **Start Game** 버튼 클릭
2. 웹캠 접근 권한 허용
3. 화면에 나타나는 과일을 손가락으로 슬라이스!

## ⚙️ 기술 스택

- MediaPipe Hands (손 인식)
- HTML5 Canvas
- Vanilla JavaScript
